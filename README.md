# Responsive Navigation Bar (HTML + CSS)

A simple and fully responsive navigation bar built using only **HTML and CSS**.  
This project demonstrates how to create a clean layout that adapts to different screen sizes without using any JavaScript.

---

## Features

- Responsive layout for desktop, tablet, and mobile
- Pure HTML and CSS (no JavaScript)
- Clean and minimal design
- Easy to customize
- Beginner friendly

---

## Technologies Used

- HTML5  
- CSS3 (Flexbox / Media Queries)

## How It Works

- HTML is used to create the structure of the navigation bar.
- CSS is used to:
  - Remove default list styles
  - Align items horizontally
  - Add colors and spacing
  - Make the layout responsive using media queries

---

## Responsive Design

The navbar adapts based on screen width:

- Desktop: Horizontal navigation menu  
- Mobile: Stacked vertical layout  

This is achieved using CSS media queries.

---

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Resize the browser window to see the responsive behavior.
